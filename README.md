# ts-dsa

使用 TypeScript 实现 《数据结构与算法分析 C语言描述——Mark Allen Weiss》书中的经典数据结构与算法:

1. 链表、栈和队列
2. 树
    - 二叉搜索树
    - AVL 树
3. 散列表
    - 分离链接法
    - 开发寻址法
4. 堆
    - 二叉堆
    - 左式堆
    - 二项队列
5. 排序
    - 插入排序
    - 希尔排序
    - 堆排序
    - 归并排序
    - 快速排序
6. 图
    - 邻接表图
    - 拓扑排序
    - 单源最短路径算法
        - 无权最短路径
        - Djikstra 算法
